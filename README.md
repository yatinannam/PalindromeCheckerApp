# Palindrome Checker App

A console-based Java application that validates whether a given string is a palindrome under different conditions. This project demonstrates core programming fundamentals and data structure concepts.

---

## Overview

The Palindrome Checker App is designed to check if strings read the same forwards and backwards, implementing various use cases to demonstrate different programming concepts in Java.

---

## Features

### UC1: Application Entry & Welcome Message

- Displays application name and version
- Initializes the system
- Provides a professional startup experience

### UC2: Print a Hardcoded Palindrome Result

- Checks a hardcoded string for palindrome properties
- Uses character-by-character comparison
- Displays the input and result

### UC3: User Input for Palindrome Check

- Accepts user input from console
- Validates the input string
- Checks if the entered string is a palindrome
- Displays the result to the user

### UC4: Case-Insensitive Palindrome Check

- Converts input to a uniform case (lowercase/uppercase)
- Ignores case differences when checking palindrome
- Example: "Madam" and "madam" both detected as palindromes

### UC5: Ignore Spaces and Special Characters

- Removes spaces from the input string
- Filters out special characters and punctuation
- Checks only alphanumeric characters for palindrome
- Example: "A man, a plan, a canal: Panama" detected as palindrome

### UC6: Multiple Palindrome Checks (Menu-Driven)

- Implements a menu-driven interface
- Allows users to check multiple strings without restarting
- Provides options to continue or exit
- Maintains session until user chooses to exit

### UC7: Exit Application Gracefully

- Provides a proper exit mechanism
- Displays a farewell message
- Closes resources properly
- Ensures clean application termination
